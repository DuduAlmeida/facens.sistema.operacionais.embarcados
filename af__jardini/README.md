## Como começar?

- Instale as dependêcias:
  ```
  npm run install-all
  ```
- Inicialize cada projeto em um terminal diferente:
  ```
  npm run turismo-front
  ```
  ```
  npm run evento-api
  ```
  ```
  npm run hotel-api
  ```
  ```
  npm run voo-api
  ```
